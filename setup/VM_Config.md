# VM Configuration

## Setup Instructions
- Use VirtualBox or VMware to create 2 VMs:
  - Ubuntu (Victim)
  - Kali Linux (Attacker)

## Networking
- Set both VMs to Internal Network or Host-only.
- Test connectivity using ping.

## Tools to Install
- Ubuntu: Python3, iptables, net-tools
- Kali: Hydra, Nmap, Netcat
